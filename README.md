# Earthly

Earthly is a web-based plant care management and tracking system designed for beginner plant owners, young adults, students, and busy individuals who want simple and structured support for taking care of indoor plants.

The system helps users explore plant information, manage a personal plant collection, track care-related activities, and maintain consistent care habits through an easy-to-use interface.

## Project Vision

Earthly focuses on simplicity, structured care routines, and personalized plant tracking. Unlike more complex plant-care platforms, Earthly is designed to give users clear guidance without overwhelming them with advanced features or unnecessary complexity.

## Main Features

### User Features
- Register a new account
- Log in securely
- Log out securely
- View a structured plant catalog
- Search plants by name or category
- Add plants to a personal collection
- Edit saved plant details such as nickname, location, and notes
- Delete plants from the personal collection
- View dashboard information with plant care guidelines and tips
- Track daily care progress using a streak feature

### Admin Features
- Log in securely
- Log out securely
- Add new plants to the catalog
- Edit existing plant information

## Project Scope

Earthly is developed as a web-based application for laptop and desktop users. The system supports a single-language interface in English and is intended to run on modern browsers.

This project does not include:
- Mobile application support
- Real-time sensor integration
- Automated plant care systems
- Expert consultation services
- Plant disease diagnosis
- E-commerce features
- Multilingual support
